Estrutura Condicional Encadeada

É uma sequência de testes lógicos, executado conforme o resultado de cada condição.

Normalmente, utilizado quando necessário verificar duas (ou mais) condições sucessivas;

Podem ter dois resultados: um para a condição verdadeira e outra para a falsa.

  • Podem ter outras condições

      • Que podem ter outras

Já no código, vamos mudar um pouco o código anterior.

Colocando uma nova condição: horasTrabalhadas

Agora, estamos trabalhando com duas condições…

pretencaoSalarial <- salario <= 2000

e

horasTrabalhadas ← //numero de horas que ele poderá trabalhar

Código

Algoritmo "EMPREGO-RH4"

Var
   horario : inteiro
   salario : real
Inicio
      escreva("Digite o salário: ")
      leia(salario)
      
      se(salario >= 2000) entao
         escreva("Candidato selecionado para próxima fase.")
      senao
         escreva("Digite a carga horária: ")
         leia(horario)
            se (horario >= 7) entao
               escreva("Candidado PRÉ-SELECIONADO para fase de condicional")
            senao
               escreva("Candidato NÃO PASSOU para a próxima fase.")
            fimse
      fimse
