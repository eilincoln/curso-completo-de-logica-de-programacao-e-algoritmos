Estrutura de dados e Arrays

Estrutura de dados e o que é

Forma de estruturar ou organizar dados na memória RAM do computador, com o objetivo de utilizar estes dados de forma mais eficiente.

Os algoritmos manipulam dados e, quando estes dados estão organizados, eles caracterizam uma forma, ou seja, uma estrutura de dados.

A organização e os métodos para manipular estas estruturas são o que otimizam e diminuem o espaço utilizado na memória RAM, além de tornar o código mais simples.

Estruturas HETEROGÊNIAS

Conjunto de dados formados por tipos de dados distintos, em uma mesma estrutura.

EX: Real, inteiro, literal e caracteres

HOMOGÊNIAS

São os conjuntos que só tem um tipo de dado, seja qual for. Só vai ter um do mesmo tipo dentro do mesmo.

Estrutura de dados

O que são arrays?

  • São estruturas de dados homogêneas, ou seja, armazenam apenas dados do mesmo tipo
  • São lineares e estáticas, isto é, possuem um número fixo de elementos de um determinado tipo de dado.
  • Estes dados são acessados individualmente por meio de um índice, onde geralmente começa por 0.
  • Dentro do array existe a matriz
