Operadores Aritméticos

São os símbolos usados para fazer matemática na programação.

Na linguagem Portugol, que é uma linguagem de programação didática utilizada no ensino de lógica de programação, os operadores aritméticos são fundamentais para realizar operações matemáticas em um programa. Esses operadores permitem que você realize cálculos e manipule valores numéricos. Vamos explorar os principais operadores aritméticos em Portugol:

  1. Adição (+):
    • Soma dois valores.
    resultado <- valor1 + valor2
  1. Subtração (-):
    • Subtrai o segundo valor do primeiro.
    resultado <- valor1 - valor2
  1. Multiplicação (*):
    • Multiplica dois valores.
    resultado - valor1 * valor2
  1. Divisão (/):
    • Divide o primeiro valor pelo segundo.
    resultado <- valor1 / valor2
  1. Resto da divisão (%):
    • Retorna o resto da divisão do primeiro valor pelo segundo.
    resultado <- valor1 % valor2
ou 
resultdo <- valor1 mod valor2
  1. Potenciação (^):
    • Eleva o primeiro valor à potência do segundo.
    resultado <- valor1 ^ valor2
  1. Incremento (++) e Decremento (--):
    • Incrementa ou decrementa o valor de uma variável em 1.
    valor++
valor--

É importante notar que a precedência dos operadores aritméticos em Portugol segue as regras matemáticas padrão. Parênteses podem ser utilizados para alterar a ordem de avaliação das expressões. Por exemplo:

resultado = (valor1 + valor2) * valor3

Lembre-se de que em Portugol, a atribuição é feita usando o operador ←. Os operadores aritméticos são frequentemente combinados com variáveis para realizar cálculos mais complexos. Exemplo:

soma <- nota1 + nota2
media <- soma / 2

Esses são os conceitos básicos dos operadores aritméticos em Portugol. Eles são essenciais para realizar operações matemáticas em programas, o que é uma parte fundamental da programação em geral.
