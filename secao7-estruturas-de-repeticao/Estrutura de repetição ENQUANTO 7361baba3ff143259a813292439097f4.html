Estrutura de repetição ENQUANTO

As estruturas de repetição, também conhecidas como loops, são utilizadas em programação para executar um bloco de código repetidamente enquanto uma condição específica for verdadeira.

O que são Estruturas de Repetição? E quais suas funções para na lógica de programação?

  • Funcionam para repetir um código, ele repete um trecho de código.
  • Ajudar no controle e na repetição de uma tarefa.
  • Sempre que necessário, repetir instruções N vezes ou até que informado “Finalizar a execução”.

Elas permitem automatizar tarefas que exigem repetição, como percorrer uma lista de elementos, processar dados ou executar operações até que uma condição seja atendida.

Os laços estudaram, serão:

  • ENQUANTO
  • REPITA
  • PARA

Mas qual eu devo usar em meu programa?

  1. Se sabemos o nº de vezes que iremos repetir ou testar um bloco de instruções, usamos PARA…REPITA-ATÉ
  1. Se NÃO sabemos o número de vezes que iremos repetir ou testar um bloco de código, usamos ENQUANTO

Vamos começar pela ENQUANTO

Se o teste lógico resultar em VERDADEIRO os comandos associados são executados. Repetindo o teste enquanto o resultado lógico for verdadeiro.

Caso seja FALSO, o programa encerra essa estrutura e volta para o fluxo normal do programa.

bastante usada em menus.

Sintaxe:

enquanto (condicao) faca {
    // Código a ser repetido enquanto a condição for verdadeira
    // ...
}

O bloco de código dentro das chaves será executado repetidamente enquanto a condição especificada dentro dos parênteses for verdadeira.
