Tipo de Identificadores

Como já foi visto, sempre precisamos declarar um tipo, sendo eles:

  • INTEIRO
  • REAL
  • CARACTERE
  • LÓGICO

Tipo Inteiro

Específicos para armazenamento de números;

Poder ser utilizada para cálculos;

Tipo de variáveis que armazena SOMENTE números INTEIROS;

Podem ser positivos ou negativos;

São os números inteiros aqueles que não tem casa decimais.

REAL

Específicos para armazenamento de números;

Poder ser utilizada para cálculos;

Tipo de variáveis que armazena números RACIONAIS ou IRRACIONAIS;

Podem ser positivos ou negativos;

CARACTERE

  • Pode ser utilizada para o armazenamento de todos os tipos de dados, caracteres ou números;
  • Não pode ser utilizada para cálculos.

Se eu for usar para calculo, tenho que definir como número.

LÓGICO

  • Específica para o armazenamento de dois valores: VERDADEIRO ou FALSO
  • Utilizada para representar variáveis que apresentam respostas;
  • Não pode ser usada para cálculos.
